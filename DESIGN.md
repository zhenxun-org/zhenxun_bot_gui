# 真寻Bot GUI 设计文档

## 设计理念

### 现代化审美标准
真寻Bot GUI 2.0 采用当前最流行的现代化设计语言，融合以下设计趋势：

- **Glassmorphism (毛玻璃拟态)**: 半透明元素和模糊效果
- **渐变美学**: 紫蓝色系渐变，营造科技感
- **卡片式设计**: 圆角卡片布局，清晰的信息层次
- **微交互**: 悬浮效果和状态变化，提升用户体验

## 色彩方案

- **占位文字**: `#a0aec0` (极浅灰)

## 组件规范

### 按钮设计
- **主要按钮**: 渐变背景 + 圆角22px + 45px高度
- **次要按钮**: 透明背景 + 边框 + 悬浮效果
- **文字规范**: 14px字体 + 600字重 + 适当内边距

### 卡片设计
- **圆角**: 12-25px (根据大小调整)
- **阴影**: 使用background-color变化替代box-shadow
- **边框**: 1px实线 + rgba白色边框
- **内边距**: 20-30px (根据内容调整)

### 字体规范
- **标题**: 20-28px + Bold权重
- **正文**: 14-16px + Medium权重  
- **说明**: 12-13px + Normal权重
- **字体族**: 系统默认字体栈

## 交互设计

### 悬浮效果
- **卡片悬浮**: 边框颜色变化 + 背景色微调
- **按钮悬浮**: 渐变色加深 + 轻微缩放效果
- **导航悬浮**: 背景透明度变化

### 状态反馈
- **选中状态**: 明显的颜色差异
- **禁用状态**: 降低透明度 + 灰色处理
- **加载状态**: 预留动画效果空间

## 响应式设计

### 最小尺寸
- **窗口最小宽度**: 1000px
- **窗口最小高度**: 700px
- **侧边栏固定**: 280px宽度
- **内容区自适应**: 剩余空间

### 滚动优化
- **自定义滚动条**: 8px宽度 + 圆角
- **滚动手柄**: 半透明蓝色 + 悬浮加深
- **滚动平滑**: 启用smooth scrolling

## 可访问性

### 对比度
- 所有文字与背景对比度 ≥ 4.5:1
- 重要信息对比度 ≥ 7:1

### 键盘导航
- 支持Tab键遍历
- 清晰的焦点指示器
- 快捷键支持

### 可读性
- 充足的行间距 (1.4-1.8)
- 合适的字体大小 (≥12px)
- 清晰的信息层次
